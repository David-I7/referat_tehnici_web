# Canvas - Aplicatii 2D

## Referat

Referatul ascoiat acestui repository se poate gasi aici: https://docs.google.com/document/d/1wLkYj_eqxtiSVuU-oxsJ497ZRFpYzAGHcQoI_AktbLU/edit?usp=sharing

## Descriere

Acest proiect conține o colecție de simple exemple simple precum si un joc 2D realizat cu **HTML5 Canvas 2D**, menit să demonstreze conceptele de bază al utilizarii canvas in browser. Exemplele ilustrează modul de utilizare al Canvas API pentru crearea de forme geometrice, animații și aplicații vizuale interactive folosind JavaScript.

## Instalare

Pentru rularea exemplelor nu este necesară instalarea unor librării sau aplicații suplimentare.

Cerințe:

- Un browser web modern (Google Chrome, Mozilla Firefox, Microsoft Edge)
- Un editor de text (opțional, pentru analizarea sau modificarea codului)

Pași:

1. Clonați repository-ul sau descărcați arhiva ZIP.
2. Extrageți fișierele (dacă ați descărcat arhiva).
3. Deschideți oricare dintre fișierele `.html` direct într-un browser web.

## Utilizare

Fiecare exemplu este implementat într-un fișier HTML separat și poate fi rulat independent.

Pentru a rula un exemplu, este suficient să deschideți fișierul corespunzător în browser.
